## How to Use This Template

1. **Fork this repo** to your own GitHub account.
    
2. Copy the **Chapter Notes Template** for each new chapter.
    
3. Fill in:
    
    - **Summary**
        
    - **Key Concepts**
        
    - **Examples & Code**
        
    - **Commands**
        
    - **Notes & Observations**
        
4. Update **Progress checkboxes** as you complete chapters.
    
5. Use the following commit convention to track your work:
    
    ```text
    chore(rustbook): complete <chapter>
    fix(rustbook): correct code example in <chapter>
    refactor(rustbook): improve notes/code clarity
    ```

### Chapter 1 â€“ Getting Started (Example)

**Summary:**  
Introduction to Rust, installing tools, and running basic programs.

**Key Concepts:**

- Rust provides modern developer tools for systems programming:
    
    - Cargo (dependency management and build tool)
        
    - rustfmt (code formatting tool)
        
    - Rust Language Server (IDE integration)
        
- Rust is beginner-friendly and suitable for students or those learning systems concepts.
    
- Syntax comparison to C.
    

**Examples:**

**C:**

```c
int main() {
    printf("Hello, world!\n");
}
```

**Rust:**

```rust
fn main() {
    println!("Hello, world!");
}
```

**Compiling & Running Programs:**

```shell
# C
$ gcc -o main main.c
$ ./main

# Rust
$ rustc main.rs
$ ./main
```

**Notes & Observations:**

- `rustup doc --book` opens the book locally.
    
- Rust ecosystem tools help maintain productivity while writing safe, systems-level code.
